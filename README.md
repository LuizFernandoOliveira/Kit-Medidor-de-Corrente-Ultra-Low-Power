# Kit Medidor de Corrente Ultra-Low Power

![Status do Projeto](https://img.shields.io/badge/Status-Desenvolvimento-yellow)
![PCB](https://img.shields.io/badge/PCB-KiCad-green)

## Descrição
Contendo quatro circuitos independentes de quatro topologias distintas, este circuito eletrônico é ideal para realizar a medição de corrente elétrica de diversas aplicação ultra-low power.

## Descrições técnicas

- Possui quatro circuitos independentes de corrente via transimpedância;
- Possui quatro circuitos independentes de corrente via rede T;
- Possui quatro circuitos independentes de corrente via High-Side;
- Possui quatro circuitos independentes de corrente via Low-Side;
- Possui alimentação de 3V3 ou 5V;
- Possui alimentação via USB tipo-C;
- Altíssima precisão e estabilidade: tensão de offset de 5 µV máx. a 25 °C, 8 µV em toda a faixa de temperatura (-40 °C a 125 °C);
- Entrada e saída rail-to-rail;
- Baixa tensão de alimentação: 1,8 - 5,5 V;
- Baixo consumo de energia: 40 µA máx. a 5 V;
- Produto ganho-largura de banda: 400 kHz;
- Alta tolerância a ESD: 4 kV HBM;
- Ampla faixa de temperatura de operação: -40 a 125 °C

## Estrutura do repositório
Os arquivos de hardware foram desenvolvidos utilizando o KiCad 9.0.
- **`.kicad_pcb`**: Layout da placa de circuito impresso (Board).
- **`.kicad_sch`**: Esquemático eletrônico do sistema (Schematic).
- **`.kicad_pro`**: Gerenciador de projeto do KiCad (Project).
